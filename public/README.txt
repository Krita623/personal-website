此目录用于存放网站所需的图片资源。
你需要在此目录中添加以下图片文件：

1. profile.jpg - 你的个人头像图片，建议尺寸为400x400像素，正方形比例
2. project-1.jpg - 项目1的展示图片，建议尺寸为800x400像素
3. project-2.jpg - 项目2的展示图片，建议尺寸为800x400像素
4. project-3.jpg - 项目3的展示图片，建议尺寸为800x400像素

你可以使用任何图片编辑工具调整这些图片的尺寸和格式。
建议使用JPG或PNG格式，以获得最佳的加载性能和图片质量平衡。

如果没有实际的项目图片，你可以使用占位图片服务，例如:
- https://placehold.co/
- https://placeholder.com/
- https://picsum.photos/
